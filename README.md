# ChoreZilla 🦖 - A Personal Task Manager

## About the Project

ChoreZilla is a full-stack web application designed to help me, Aaron (username CommanderThrow), organize my daily tasks. This project serves as my first major step into the world of full-stack software engineering, applying the fundamentals of front-end, back-end, and database development. It's built to be a simple, yet robust, personal tool to manage my own tasks, while also showcasing my ability to build a complete web application from the ground up.

## Features ✨
### User-facing Features
**Task Management:** Users can create, read, update, and delete tasks. Each task has a title (required), an optional description, and an optional due date.

**Task Status:** Tasks can be marked with one of three statuses: "To Do," "In Progress," or "Completed." This allows for easy tracking of progress.

**Filtering and Sorting:** The task list can be filtered by status and sorted by due date, making it easy to find and prioritize tasks.

**User Authentication:** ChoreZilla includes a basic user authentication system, ensuring that each user's tasks are private and secure.

**Responsive Design:** The application is designed to be fully functional on both desktop and mobile devices.

### Technical Features
**Full-Stack Architecture:** The application is built with a clear separation between the front end and the back end, communicating via a RESTful API.

**Clean Code:** The codebase is well-organized and includes comments to explain complex logic.

**Version Control:** The entire project is managed using Git, with a clear commit history that documents the development process.
